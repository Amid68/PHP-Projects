# PHP-Number-Guessing

## Overview
This project, part of the Codecademy's Learn PHP course, involves creating a number guessing game in PHP. The program generates a random number between 1 and 10, and the user tries to guess this number. The game is run 10 times, and after the rounds, the user is given feedback on their guessing abilities.

## Features
- **Random Number Generation**: Each round, the program generates a new random number between 1 and 10.
- **User Guesses**: The user is prompted to guess the number in each round.
- **Feedback**: After each guess, the program informs the user whether their guess was too high, too low, or correct.
- **Guess Statistics**: At the end of 10 rounds, the program provides statistics on the user's performance, including the percentage of correct guesses and a tendency to guess high or low.

## How to Run
1. Ensure PHP is installed on your system.
2. Run the script in a PHP-supported command-line environment.
3. Follow the on-screen prompts to make guesses.

Enjoy testing your intuition with this simple yet engaging number guessing game in PHP!
